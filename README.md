# mbtiles2ungzpbf

A very simple project to convert an mbtiles file into a set of ungzipped pbf tiles for local serving.

## Install

With [npm](https://npmjs.org):

```
$ npm install mbtiles2ungzpbf -g
```

## Usage

```
$ mbtiles2ungzpbf -f tiles.mbtiles
```
